# 🎬 Netflix Movies & TV Shows Data Analysis

## 📌 Project Overview

This project analyzes the Netflix Movies & TV Shows dataset using Python, Pandas, NumPy, Matplotlib, and Seaborn. The objective is to explore content trends, identify patterns, visualize key findings, and generate business insights from the data.

---

## 🎯 Project Objectives

* Load and inspect the Netflix dataset
* Clean and preprocess the data
* Perform Exploratory Data Analysis (EDA)
* Create visualizations to identify trends and patterns
* Generate business insights and recommendations

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook
* VS Code

---

## 📂 Dataset

Netflix Movies & TV Shows Dataset from Kaggle.

Dataset includes:

* Title
* Type (Movie / TV Show)
* Director
* Cast
* Country
* Release Year
* Rating
* Duration
* Date Added

---

## 📊 Exploratory Data Analysis

The following questions were investigated:

1. Which type of content is more common on Netflix?
2. Which countries contribute the most content?
3. How has Netflix content grown over time?
4. What are the most common audience ratings?
5. Which directors appear most frequently?

---

## 📈 Visualizations

The project includes:

* Bar Chart
* Histogram
* Pie Chart
* Line Chart
* Scatter Plot
* Heatmap

---

## 🔍 Key Insights

1. Movies significantly outnumber TV Shows in Netflix's catalog.
2. Netflix experienced rapid content growth after 2015.
3. Content production is concentrated among a small group of countries.
4. Teen and adult audiences represent the primary target demographic.
5. Netflix strongly emphasizes recent and contemporary content.

---

## 💡 Most Surprising Finding

The most surprising discovery was the sharp increase in content additions after 2015. The data revealed a dramatic expansion period that highlights Netflix's aggressive growth strategy and increasing investment in content production.
